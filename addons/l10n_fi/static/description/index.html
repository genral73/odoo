Features

New Fields for Partners / Contacts:

  • Business ID (Y-tunnus)
  • eInvoicing Address (verkkolaskuosoite)
  • eInvoicing Operator (verkkolaskuvälittäjä)
You can also define your company's eInvoicing address and operator from the Accounting Settings.

The eInvoicing operators can be managed from Accounting -> Configuration -> E-Invoicing

Invoice Payment Reference Types:

  • Finnish Standard Reference
  • Finnish Creditor Reference (RF)
Set the payment reference type from the Sales Journal.
